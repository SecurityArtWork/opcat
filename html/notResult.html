Ayuda

Para proceder a la obtención de registros tendrá que realizar una serie de pasos que le indicamos a continuación:

  1. Iniciar el script localizado en el proyecto y accesible mediante la ruta 'scanner/app/opcat.py'.Por defecto la herramienta escanea direcciones IP aleatorias y terminará cuando se encuentre un servicio abierto en el puerto 80. Para ver las diferentes opciones se puede ejecutar el script con la opción -h.
  2. El script debe ser iniciado en un sistema Linux que cuente con interfaz gráfica y Firefox como explorador web instalado.
  3. Una vez se inicie el script debe mantenerse a la espera de descubrir servicios web accesibles.
  4. Tras descubrir al menos un servicio podrá recargar la página y navegar por los resultados obtenidos.
